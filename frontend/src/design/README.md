# Design Tokens

This folder contains shared design tokens for the frontend.

Use these values when building components so every teammate creates the same visual style.

Files:

- `tokens.css`: CSS custom properties for styling.
- `tokens.ts`: TypeScript constants for component libraries, chart configs, and inline configuration where CSS variables are not enough.

Do not create new colors, spacing values, radii, or shadows inside a page without updating the design system docs.
